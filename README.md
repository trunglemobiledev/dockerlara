# Docker for laravel

### Clone source
```
$ git clone https://github.com/trunglemobiledev/dockerlara.git
```

### Copy docker-compose
```
$ copy docker-compose.yml back to folder.
```

#### build docker
```
$ docker build ./docker
```
#### or use docker compose
```
$ docker-compose build
```
#### run docker
```
$ docker-compose up
```

#### delete image docker
```
$ docker system prune -a
```

#### check container docker
```
$ docker ps -all
```
